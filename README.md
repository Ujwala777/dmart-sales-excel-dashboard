# DMart Sales Excel Dashboard

## 📌 Project Overview

This project presents an interactive Excel dashboard created to analyze DMart sales performance across product categories, regions, cities, and payment methods. The dashboard transforms raw sales data into meaningful visual insights that help understand sales patterns and business performance.

## ⚠️ Problem Statement

Retail businesses generate large volumes of sales data including product categories, regions, payment methods, and city-wise transactions. Analyzing this raw data manually makes it difficult to identify sales trends, top-performing categories, and regional performance.

This project solves the problem by building an interactive Excel dashboard that converts raw sales data into clear visual insights for easier analysis and better decision-making.

## 🎯 Objectives

* Analyze total sales across different product categories
* Compare sales performance by region (East, West, North, South)
* Identify top-performing product categories
* Analyze customer payment methods (Cash, Credit Card, Debit Card, UPI)
* Evaluate city-wise sales performance
* Understand product-wise sales trends

## ❓ Business Questions

* Which product category generates the highest sales?
* Which region contributes the most revenue?
* Which cities generate the highest sales?
* Which payment method is most frequently used by customers?
* Which products contribute the most to total sales?
* How do sales vary across different regions?

## 📊 Dashboard Features

* **Category-wise Sales Analysis** – Comparison of sales across Clothing, Electronics, Furniture, and Groceries
* **Regional Sales Distribution** – Sales contribution from East, West, North, and South regions
* **Payment Method Analysis** – Sales comparison using Cash, Credit Card, Debit Card, and UPI
* **City-wise Sales Analysis** – Sales performance across different cities
* **Product-wise Sales Trends** – Analysis of sales by product names
* **Interactive Filters** – Region filters using slicers for dynamic analysis
